Game Instructions

Memory Game

  • A matching pair is worth 3 points.
  • A mismatched pair subtracts 1 point.
  • Score 10 or more points to level up!!!

Total Score: 0



Go ahead and click (flip) the cards until you find a match. Once you do all the cards shuffle around!

Intense Gameplay!

What makes this game so exciting is that it pits you against the most challenging foe ever faced, yourself!
